## 1. Introduction to UAVs
A **Unmanned Aerial Vehicle (UAV)** is more than just a flying machine; it is a component of a **UAS (Unmanned Aerial System)**.

| Term | Scope |
| :--- | :--- |
| **UAV** | The physical aircraft/drone. |
| **UAS** | The UAV + Ground Control Station (GCS) + Data Links + Software. |

## 3. Types of Drones & Applications

### üåÄ Multirotor
* **Design:** Quadcopters, Hexacopters.
* **Best For:** Precise hovering, photography, and indoor flight.
* **Limit:** Low battery endurance (15‚Äì40 mins).

### ‚úàÔ∏è Fixed-Wing
* **Design:** Airplane-like wings.
* **Best For:** Mapping vast areas, long-range surveillance.
* **Limit:** Requires a runway or launcher; cannot hover.

### üîÅ VTOL (Vertical Take-Off & Landing)
* **Design:** Hybrid (Multirotor + Fixed-wing).
* **Best For:** Long-range missions starting from confined spaces.

## 4. Core System Components
A drone is an integrated **cyber-physical system**:

1.  **Flight Controller (FC):** The brain (runs ArduPilot/PX4). Uses IMUs for stability.
2.  **Propulsion:** BLDC Motors + ESCs + Propellers.
3.  **Power:** LiPo/Li-ion batteries + Power Distribution Boards (PDB).
4.  **Sensors:** GPS (Navigation), Barometer (Altitude), LiDAR (Obstacle avoidance).
5.  **Payload:** Gimbals, Cameras, Sprayers, or Cargo hooks.
